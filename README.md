# Omato-Delivery-Dashboard-

🚀 Project Overview

The Omato Delivery Dashboard helps track key business metrics such as orders, revenue, delivery performance, customer trends, and regional insights. It also includes a dedicated Bottom Analysis Dashboard for deeper analytics.

🔍 Key Features
Order Performance Tracking
Revenue & Sales Analysis
Delivery Time & Status Monitoring
Customer Trend Insights
Region-wise Performance Analysis
Interactive Filters & Slicers
KPI Cards
Multi-page Navigation Buttons
🛠️ Tools Used
Power BI
Power Query
DAX
Data Cleaning
Data Modeling
📈 Business Insights

This dashboard transforms raw delivery data into meaningful insights for faster and smarter decision-making.
