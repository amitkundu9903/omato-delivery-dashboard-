# Omato-Delivery-Dashboard-
Omato Delivery Dashboard is an interactive Power BI project analyzing orders, revenue, delivery status, customer trends, and regional performance. Features include KPI cards, dynamic filters, and page navigation. Built using Power BI, Power Query, and DAX for business insights and decision-making.
